# Chess Tutor: Adaptive Teaching with Probabilistic ML

> **STA 561D — Probabilistic Machine Learning, Duke University**
>
> An adaptive chess tutoring system that predicts human moves at any ELO level, provides skill-appropriate feedback, and uses contextual Thompson Sampling to select optimal teaching strategies.

Group members: Yushu Lei, Jiahe Li, Ziyang Qin, Yiming Cui, Zherui Zhang
